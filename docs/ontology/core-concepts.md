# Core Concepts

This page defines the fundamental concepts used in the system.

## Entity

An **Entity** is any uniquely identifiable object within the system.

Example:

- User
- Document
- Transaction

## Actor

An **Actor** is an entity capable of initiating actions.

Examples:

- Human user
- Automated service

## Resource

A **Resource** is an entity that can be created, read, updated, or deleted.

Notes:

- All resources are entities
- Not all entities are resources
